Ingredientes das Poções

Os ingredientes são essenciais para criar poções no Minecraft. Cada ingrediente possui uma função específica, podendo criar uma nova poção ou modificar os efeitos de uma já existente.

Ingrediente Função
Verruga do Nether Base para a maioria das poções.
Pó de Blaze Combustível utilizado no suporte de poções.
Melancia Reluzente Usada para criar a Poção de Cura.
Lágrima de Ghast Usada para criar a Poção de Regeneração.
Creme de Magma Usado para criar a Poção de Resistência ao Fogo.
Cenoura Dourada Usada para criar a Poção de Visão Noturna.
Pé de Coelho Usado para criar a Poção de Salto.
Açúcar Usado para criar a Poção de Agilidade.
Olho de Aranha Usado para criar a Poção de Veneno.
Olho de Aranha Fermentado Modifica algumas poções e pode criar efeitos negativos, como Fraqueza.
Peixe Baiacu Usado para criar a Poção de Respiração Aquática.
Membrana de Fantasma Usada para criar a Poção de Queda Lenta.
Pó de Redstone Aumenta a duração das poções.
Pó de Glowstone Aumenta o nível (potência) da maioria das poções.
Pólvora Transforma uma poção comum em uma Poção Arremessável.
Bafo de Dragão Transforma uma Poção Arremessável em uma Poção Persistente.
